# Ejercicios por funcionalidad

## 1) Valores de Tendencia Central y Dispersión

1. Ingresa datos muestrales: `12, 15, 14, 10, 18, 12, 16`. Interpreta media, mediana, moda, varianza y desviación estándar.  
2. Carga un CSV con una columna “ventas_mensuales” y calcula las métricas poblacionales; compara con el cálculo muestral.  
3. Ajusta la lista para incluir un valor atípico (p. ej. reemplaza 18 por 40) y observa cómo cambian media y desviación estándar.

## 2) Valor Z

1. Calcula el valor Z para x = 72, μ = 65, σ = 5. Interpreta cuántas σ está x por encima de la  media.
2. A partir de una probabilidad acumulada p = 0.975, obtén el Z correspondiente y relaciona con un nivel de confianza del 95%.  
3. Compara dos valores: x₁ = 58 con μ = 50, σ = 4 y x₂ = 120 con μ = 100, σ = 15. ¿Cuál está relativamente más lejos de su media?

## 3) Tamaño de Muestra

1. Proporción: estima p con margen de error 4% y confianza 95%, sin proporción previa (usa p = 0.5).  
2. Media: con σ estimada de 12 unidades, margen de error 3 unidades, confianza 99%. Calcula el tamaño necesario.  
3. Repite el ejercicio 1 suponiendo población finita de 5000 y compara con el tamaño para población infinita.

## 4) Intervalo de Confianza para Una Población

1. Media con σ conocida: x̄ = 25.2, σ = 0.5, n = 50, confianza 95%.  
2. Media con σ desconocida (usar t): x̄ = 18.4, s = 2.1, n = 16, confianza 90%.  
3. Proporción: 280 éxitos de 400, confianza 95%. Interpreta el intervalo.

## 5) Comparación entre Dos Poblaciones

1. Dos medias (σ desconocida, usar t): Grupo A (n=22, x̄=68, s=6), Grupo B (n=20, x̄=64, s=5). Confianza 95%.  
2. Dos proporciones: p₁ con 90 éxitos de 150, p₂ con 70 de 140. Confianza 95%. Concluye si difieren.  
3. Cambia a confianza 99% y observa cómo se ensancha el intervalo/diferencia.

## 6) Error Estándar

1. Calcula el error estándar de la media para s = 4 con tamaños de muestra n = 10, 40, 160. Observa cómo decrece.  
2. Calcula el error estándar de la proporción para p = 0.3 con n = 50, 200, 800.  
3. Relaciona el error estándar obtenido con el ancho esperado de un intervalo de confianza al 95%.

## 7) Cálculo t

1. Obtén el valor t crítico para α = 0.05 (bilateral) con gl = 12.  
2. Para α = 0.01 (cola derecha) con gl = 25.  
3. Compara el t crítico con el z crítico cuando gl es grande (p. ej. gl = 200).

## 8) Prueba de Hipótesis

1. Media, cola izquierda: H0: μ = 100, n = 36, x̄ = 96, s = 10, α = 0.05. Decide y concluye.  
2. Proporción, bilateral: H0: p = 0.3, n = 200, éxitos = 75, α = 0.05. Calcula z, valor p y decisión.  
3. Dos medias (bilateral): H0: μ₁ = μ₂. Grupo 1 (n=30, x̄=52, s=8), Grupo 2 (n=28, x̄=48, s=7), α = 0.05. Determina si hay diferencia significativa.
